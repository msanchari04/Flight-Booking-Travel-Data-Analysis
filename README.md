Flight-Booking-Travel-Data-Analysis
Analysed Flight booking dataset of 251 Passenger records across 9 attributes to uncover insights on revenue 
performance, travel class preference, pricing behaviour, passenger demographics and payment trends. 
Conducted KPI based analysis on Booking Volume, Revenue, travel class, distance-pricing Correlation and 
passenger demographics to assess Business performance. 
Implemented POWERBI dashboard using DAX(SWITCH, Calculated Columns), Star-Schema Modeling 
and Exploratory Data Analysis Techniques. 
Delivered insights that supported Premium class demand strategy, Airline Revenue Optimization, targeted 
marketing by age group and data-driven pricing and route planning Decisions.
